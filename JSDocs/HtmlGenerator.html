Class: HtmlGenerator

HtmlGenerator(rng)

This is the class to be used by clients to generate HTML tables that try, to the greatest extent possible, to replicate the contents and appearance of the Google Sheets range they are taken from. Currently, assumes that the first row of the given range is to be converted to HTML table td elements and the that the remaining rows correspond to tr HTML table rows.
